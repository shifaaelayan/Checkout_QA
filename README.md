# Selenium Checkout Automation

This project is an end-to-end automation test using Selenium WebDriver and TestNG
on the SauceDemo website.

## What I Learned
- Automating a complete user flow (Login → Add to Cart → Checkout)
- Using Selenium WebDriver to locate elements and perform actions
- Applying TestNG annotations and test priorities
- Structuring test cases in a logical execution order
- Handling form inputs and navigation during checkout

## Test Flow
1. Login with valid credentials
2. Add multiple products to the cart
3. Navigate to cart and start checkout
4. Fill checkout information
5. Complete the order successfully

## Tools Used
- Java
- Selenium WebDriver
- TestNG
- EdgeDriver
